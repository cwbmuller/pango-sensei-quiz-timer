== Description ==
= Description =
Add an options timer to Sensei Quizzes.

= Details =
* Used with WooThemes [Sensei](http://www.woothemes.com/products/sensei/) LMS Plugin
* Add an optional timer to a Sensei Quiz
* When the user has less than 5 min left timer turns orange
* When the user has less than 1 min left timer turns red
* Quiz automatically submitted when timer reaches 0

== Installation ==

1. Upload 'plugin-name.php' to the '/wp-content/plugins/' directory
,
2. Activate the plugin through the 'Plugins' menu in WordPress.
,
3. In Dashboard, navigate to Sensei > Settings > Quiz Timer and select your required settings
,
4. Enjoy the quiz timer :)
